# Your Desk Pet

Your Desk Pet is a lightweight productivity companion that places a virtual pet on your desktop during focused work sessions. The pet rewards disciplined focus and reacts (with humor and judgment) when you get distracted.

Getting started (MVP)
1. Clone the repo
2. Install dependencies:
   - npm install
3. Start in dev:
   - npm run dev
4. Build:
   - npm run build

MVP Features
- Configurable focus/break timers
- Always-on-top pet overlay
- Basic active-window distraction detection
- Pet mood changes and break-time rewards

Privacy
- Local-only detection by default (no data leaves your machine).
- For robust website detection we recommend using an optional browser extension that communicates locally.

Roadmap
- Browser extension for reliable site detection
- More personalities and animations
- Packaging for macOS, Windows and Linux
